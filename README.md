## 緣起

我大學念的是機械系，真正開始工作後，因為接觸到 BSP（Board Support Package），才逐步接觸到 Linux Kernel、驅動程式等底層領域。這些內容過去一直是我似懂非懂的領域，因此一直希望能把底層系統好好補強起來。

一次偶然的機會，看到了 NCTU OSDI 這門課的教材與作業都公開在網路上。內容完整、循序漸進，對於想自學作業系統的人來說非常珍貴。於是我決定邊做邊學，希望能藉由實作的過程，把自己對作業系統的理解真正打穩。

截至目前，我已經完成了大部分的 required requirements，以及部分的 elective。
能走到這裡，也很感謝 2025 年這個時代有 AI 的協助：
在我卡住、看不懂、找不到方向時，AI 不只幫我解析題意，還會提供我可能的設計方向與實作方式，讓我能更穩地向前推進。

期望自己繼續努力，把剩餘未完成的部分逐一補齊，希望自己能真正掌握 OS 的核心概念。
